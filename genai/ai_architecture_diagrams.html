1a

Price Signal Forecasting — LMP Prediction

Predicts Locational Marginal Prices (LMP) at key nodes across PacifiCorp's WECC footprint. Raw ISO, weather, and gas data flows from Databricks Volumes through a multi-layer feature pipeline into parallel per-node ML models. The full forecasting chain is packaged and deployed as a production agent using Mosaic AI Agent Bricks, with real-time inference served via Mosaic AI Model Serving endpoints to dispatch and trading teams.

1b

Demand Signal Forecasting — Load Prediction

Predicts hourly electrical load (MW) across PacifiCorp's 6-state service territory at multiple horizons. AMI smart meter interval data in Databricks Volumes feeds a hierarchical Many Models Forecasting pipeline (feeder → zone → system) incorporating EV charging profiles, behind-the-meter solar, and weather drivers. The demand forecasting agent is deployed via Mosaic AI Agent Bricks (Knowledge Assistant) as a versioned endpoint consumed by dispatch and the Price/Congestion models.

Congestion & Transmission Signal Forecasting

Three-stage pipeline predicting which transmission constraints will bind, their shadow prices, and node-level LMP congestion components. Uses shift factor matrix, outage schedules, and demand forecasts. The full congestion pipeline is orchestrated and deployed as a production agent using Mosaic AI Agent Bricks, with outputs feeding FTR valuation, dispatch optimization, and PSPS wildfire planning.

ELC — AI-Powered Personalized Beauty Advisor (GCP)

Conversational AI system replicating the luxury in-store consultant experience digitally — deployed across ELC brand websites. A LangGraph stateful conversation graph orchestrates a multi-turn dialogue: an Intent Parser node extracts preferences, a RAG Retriever node calls Vertex RAG Engine + Retail Search as LangChain tools, and a Response Generator node produces on-brand Gemini responses. Loop edges handle follow-ups naturally. Live as Jo Malone London AI Scent Advisor.
